## HTML-Related Extraction

This folder collects the project files that are directly related to the HTML front end.

Included:

- `Simulation/` HTML prototype pages and their CSS/JS dependencies
- `traffic_ai/web/templates/` Flask HTML templates
- `traffic_ai/web/static/` Flask CSS/JS assets
- `traffic_ai/web/app.py` because it wires the templates and static files into the Flask app

Notes:

- Original files were copied, not moved or deleted.
- Virtual environment files and unrelated dependencies were intentionally excluded.
